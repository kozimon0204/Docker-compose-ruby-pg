Docker-compose-ruby-postgres
====

Overview

Dockerとdocker-composeを使ってRuby、PostgreSQLの環境を作成する

Create a Ruby 2.5.0 and PostgreSQL environment with Docker and docker-compose.

## Description
Docker 17.12.0

docker-compose 1.19.0

Ruby 2.5.0

PostgreSQL
